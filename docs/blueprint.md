# **App Name**: CampusConnect

## Core Features:

- Homepage with Announcements: Display key information about the college such as latest events, news, and deadlines.
- Departments Showcase: A page for each academic department, showing faculty, courses offered, and research areas.
- Faculty Directory: Searchable directory of faculty members with contact information and areas of expertise.
- Admissions Information: Details for prospective students regarding admissions procedures, eligibility criteria, and important dates. An AI tool recommends relevant programs for a prospective student given their academic interests and history.
- Events Calendar: A calendar displaying academic and extracurricular activities on campus.
- Contact Information: Centralized page for contacting different departments and key personnel.
- Image Gallery: Photo gallery showcasing campus life, events, and facilities.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) to evoke trust, stability, and knowledge.
- Background color: Very Light Gray (#F9FAFB) to provide a clean and modern feel.
- Accent color: Soft Yellow (#FACC15) to highlight important elements and calls to action.
- Body and headline font: 'PT Sans', a modern, humanist sans-serif font that provides excellent readability on screen.
- Use simple, professional icons that align with the college's branding and enhance user navigation.
- Maintain a clean, grid-based layout for consistent presentation across all pages. Ensure the site is fully responsive and accessible on all devices.
- Use subtle animations and transitions to enhance the user experience without being distracting. Examples include loading animations and hover effects.
